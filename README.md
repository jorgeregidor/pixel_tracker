# Pixel_tracker


Is an application that create pixel trakers and check them:

## Installation

### Characteristics

* Ruby 2.3.1
* Rails 5.1.6
* SQLite
* Rspec 3.8
* Boostrap v4.1.3


## Usage

```
$ git clone https://github.com/jorgeregidor/pixel_tracker
$ cd pixel_tracker

$ bundle install
$ rails db:create
$ rails db:migrate

```

## Run
```
$ rails s
```
Open "localhost:3000"

## Test

```
$ bundle exec rspec
```
